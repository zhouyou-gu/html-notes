Welcome Note

This repository is a no-build collection of public HTML notes. Each note is a standalone page under notes/, listed in notes.json, and linked from the homepage.

May 26, 2026

How updates work

Add a focused HTML page under notes/, update notes.json, and add the link to index.html. Keep URLs stable when editing existing notes.

Privacy rule

Treat every note as public. Before publishing, remove credentials, private conversations, personal identifiers, local paths, and confidential third-party material. If the content is ambiguous, ask before publishing.
